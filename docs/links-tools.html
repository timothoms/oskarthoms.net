Tools

 

Unless otherwise noted, software listed here is open source and free. On macOS, most can be installed and maintained with Homebrew.

 

Creating documents

  • The LaTeX Project (typesetting)
  • macTeX (LaTeX distribution for macOS)
  • The Not So Short Introduction to LaTeX2ε (.pdf)
  • formatting information: A beginner’s introduction to typesetting with LaTeX (book)
  • Pandoc (powerful document converter)
  • Bookends (good, customizable reference manager for macOS, neither open source, nor free)
  • Zotero (reference manager)
  • biblatex (state-of-the-art citations in LaTeX)

Notes & Markdown

  • Agenda (notes linked to dates, supports Markdown, excellent, neither open source, nor free for premium features)
  • Simplenote (notes synced across devices, supports Markdown)
  • nvAlt (very fast notes app for macOS)
  • The Plain Person’s Guide to Plain Text Social Science
  • CommonMark (Markdown specification)
  • MultiMarkdown (Markdown specification)
  • Pandoc’s Markdown (Markdown specification)
  • RMarkdown

Editors & version control

  • Sublime Text [official documentation] (awesome editor, neither open source, nor free)
  • Atom
  • Colon
  • Virtual Studio Code
  • Zettlr (IDE for writing)
  • Modern CSV (excellent, fast CSV editor, neither open source, nor free for premium features)
  • git (version control)
  • GitHub (version control collaboration with yourself and others)

macOS

  • Homebrew (package manager)
  • Objective-See (security tools)
  • Mac Backup Guru (bootable backups)
  • OnyX (multifunction utility)
  • CheatSheet (help with keyboard shortcuts)
  • Scheduler
  • f.lux (adjust screen colour according to time of day)
  • Rectangle (window management)

Security

  • pwSafe (password manager)
  • Cryptomator (file encryption)
  • OnionShare (secure and anonymous sharing)

Web (including development)

  • Firefox (my preferred everyday browser)
  • Chromium (my preferred browser for certain tasks, including Google Sheets, Developer tools, etc)
  • W3Schools (web technologies reference)
  • Jekyll (static website generator)
  • Hugo (static website generator)
  • Bootstrap (front-end web framework)
  • Gatsby (front-end web framework)
  • Cyberduck (cloud storage browser, i.e. FTP etc)
  • Colorpicker

PDF

  • Skim (PDF reader and annotator)
  • Tabula (PDF table extraction)

Other

  • Syncthing (continuous file synchronization)
  • Clocker (timezone tool for macOS)
  • Kap (screen recordings)
